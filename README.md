# ChuckJokesVue

A Vue 3 application that displays Chuck Norris jokes as Bootstrap cards.

## Activity 12

- Vue 3 instance via CDN
- Chuck Norris joke data rendered with `v-for`
- Custom dark-theme CSS stylesheet

## Homework 10

- `chuck-card` reusable component with `icon_url` and `value` props
- Bootstrap 5 grid: **3 cards per row** on medium+ screens, 1 card per row on mobile
- Hover animations and responsive layout

## Project Structure

```
index.html      ← main app + chuck-card component
css/style.css   ← custom dark theme styles
README.md
```

## Running locally

Open `index.html` in a browser (or serve via XAMPP/WAMP/MAMP/Live Server).
No build step required — Vue 3 and Bootstrap 5 load from CDN.
